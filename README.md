## 如何选择机场

### 1. 看线路
- **Delay**: 延迟低的线路体验更好
- **Bandwidth**: 带宽越大越好
- **Traffic**: 流量是否够用

### 2. 看协议
- **SS/SSR**: 经典协议，兼容性好
- **V2Ray (VMESS)**: 抗封锁能力强
- **Trojan**: 新协议，速度快
- **WireGuard**: 未来趋势

### 3. 看面板
- 支持一键订阅
- 支持客户端导入
- 有测速功能
