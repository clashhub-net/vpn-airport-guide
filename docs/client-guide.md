# 客户端配置教程

## Windows

### Clash for Windows
1. 下载 [Clash for Windows](https://clash-for-windows.net)
2. 打开软件，点击「配置」
3. 粘贴订阅链接
4. 点击「代理」选择节点
5. 开启系统代理

### v2rayN
1. 下载 [v2rayN](https://github.com/2dust/v2rayN/releases)
2. 订阅管理 → 添加订阅
3. 粘贴机场提供的订阅地址
4. 更新订阅
5. 选择节点 → 启动

## macOS

### Clash Verge
1. 下载 [Clash Verge](https://github.com/clash-verge-rev/clash-verge-rev/releases)
2. 导入订阅或配置
3. 选择代理节点

### V2rayU
1. 安装 [V2rayU](https://github.com/yanue/V2rayU/releases)
2. 订阅管理添加机场订阅
3. 启动代理

## iOS

### Shadowrocket（小火箭）
1. App Store 下载
2. 点击「+」→ 订阅
3. 粘贴订阅链接
4. 启用代理

### Quantumult X
1. 下载 Quantumult X
2. 配置文件 → 引用
3. 添加订阅

## Android

### Clash for Android
1. 下载 [Clash for Android](https://github.com/Kr328/ClashForAndroid/releases)
2. 导入配置或订阅
3. 启动代理

### v2rayNG
1. 下载 [v2rayNG](https://github.com/2dust/v2rayNG/releases)
2. 订阅 → 添加
3. 更新订阅
4. 选择节点启动
